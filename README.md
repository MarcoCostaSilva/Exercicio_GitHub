# Projeto Login Dashboard

## Descrição
Projeto de login com tema escuro, validação em JavaScript, dashboard e cadastro de usuários.

## Estrutura
- index.html → Página de login
- style.css → Estilo da página
- script.js → Validação do login
- cadastro.html → Formulário de cadastro
- dashboard.html → Painel do usuário

## Como testar
1. Abrir `index.html` no navegador.
2. Preencher usuário e senha:
   - Usuário: `adm`
   - Senha: `123`
3. Testar mensagens de erro para campos vazios ou login incorreto.
